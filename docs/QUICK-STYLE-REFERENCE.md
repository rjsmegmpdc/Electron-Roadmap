var(--bg-primary);\n  padding: var(--spacing-lg);\n  border: 1px solid var(--border-color);\n  border-radius: var(--radius-md);\n  box-shadow: var(--shadow-md);\n  transition: var(--transition-colors);\n}\n\n.my-component:hover {\n  background: var(--bg-hover);\n  box-shadow: var(--shadow-lg);\n}\n```\n\n### Responsive Design\n```css\n/* Desktop (default) */\n.my-component {\n  display: grid;\n  grid-template-columns: 1fr 1fr 1fr;\n}\n\n/* Tablet */\n@media (max-width: 1024px) {\n  .my-component {\n    grid-template-columns: 1fr 1fr;\n  }\n}\n\n/* Mobile */\n@media (max-width: 768px) {\n  .my-component {\n    grid-template-columns: 1fr;\n  }\n}\n```\n\n### Dark Theme\n```css\n/* Light theme (default) */\n.my-component {\n  color: var(--text-primary);\n  background: var(--bg-primary);\n}\n\n/* Dark theme - automatic via CSS variable override */\n@media (prefers-color-scheme: dark) {\n  :root {\n    --text-primary: #FFFFFF;\n    --bg-primary: #2D2D2D;\n  }\n}\n```\n\n## Color Combinations\n\n### Text on Background\n- Dark text (`--text-primary`) on light backgrounds (`--bg-primary`)\n- Light text (`--text-inverse`) on dark backgrounds or colored elements\n- Secondary text (`--text-secondary`) for less important content\n\n### Status Indicators\n- Success: `--color-success` (#00A45F)\n- Warning: `--color-warning` (#FF9500)\n- Error: `--color-error` (#E74C3C)\n- Info: `--color-info` (#3498DB)\n\n## Best Practices\n\n✅ **Do:**\n- Use `var(--variable-name)` for all design tokens\n- Follow the class naming conventions\n- Keep all styles in `unified-styles.css`\n- Test responsive behavior\n\n❌ **Don't:**\n- Hardcode color values (use CSS variables)\n- Create inline styles in React\n- Add new CSS files\n- Use unclear class names\n\n## File Location\n\n**Main Stylesheet**: `app/renderer/styles/unified-styles.css`  \n**Imported in**: `app/renderer/App.tsx`\n\n## Documentation\n\n- **Full Reference**: Verify CSS file is imported in `App.tsx`\n- Clear browser cache\n- Check browser DevTools\n\n**Variable undefined?**\n- Use correct variable name with `--` prefix\n- Check `:root` section in
