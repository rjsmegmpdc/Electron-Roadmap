How to Test

  1. Create a test CSV file:
    • Click "Download Sample CSV" to get a test file, OR
    • Copy the sample content above into a new file and save as .csv
  2. Test the import:
    • Click "Open Timeline & Test Import"
    • On the timeline page, click "Import CSV" button
    • Select your CSV file
    • Confirm the import when prompted
  3. Verify results:
    • Check that new projects appear on the timeline
    • Open browser console (F12) to see detailed import logs
    • Test export to verify the imported data is correct
